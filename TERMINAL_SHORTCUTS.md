# Terminal Shortcuts Guide (Cursor/VS Code Terminal)

## Copy & Paste in Terminal

**Important:** `Ctrl+C` in terminal means **"Cancel/Interrupt"** (kills the current command), NOT "Copy"!

### ✅ Correct Copy/Paste Shortcuts:

**Linux/Ubuntu (Cursor Terminal):**
- **Copy:** `Ctrl+Shift+C` (or right-click → Copy)
- **Paste:** `Ctrl+Shift+V` (or right-click → Paste)

**macOS:**
- **Copy:** `Cmd+C` (works in terminal for copying)
- **Paste:** `Cmd+V`

**Windows:**
- **Copy:** `Ctrl+Shift+C` 
- **Paste:** `Ctrl+Shift+V`

## Alternative Methods:

1. **Select Text:** Click and drag to select text in terminal
2. **Right-click:** Right-click the selection → Copy
3. **Right-click empty space:** Right-click → Paste (if you have something copied)

## Other Useful Terminal Shortcuts:

- `Ctrl+C` - Stop/cancel current command
- `Ctrl+D` - Exit terminal / End of file
- `Ctrl+L` - Clear terminal screen
- `Ctrl+U` - Clear line from cursor to start
- `Ctrl+K` - Clear line from cursor to end
- `Ctrl+A` - Move cursor to start of line
- `Ctrl+E` - Move cursor to end of line
- `Ctrl+R` - Search command history (reverse search)
- `Tab` - Auto-complete file/command names
- `Up/Down Arrow` - Navigate command history

## To Copy Error Messages:

1. Select the error text with your mouse
2. Press `Ctrl+Shift+C` (or right-click → Copy)
3. Paste it somewhere with `Ctrl+Shift+V`
